# Dados 3D para GitHub Pages

Aplicación web de dados aleatorios con animación 3D.

## Funciones

- Dados D4, D6, D8, D10, D12 y D20.
- Agregar y eliminar dados.
- Cambiar individualmente el tipo de cada dado.
- Lanzar todos o solo uno.
- Bloquear dados para conservar su resultado.
- Animación 3D.
- Suma total, valor mayor y menor.
- Historial de hasta 30 lanzamientos.
- Tema claro y oscuro.
- Guarda el estado automáticamente en el navegador.
- Compatible con celular y computador.

## Cómo subirlo a GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube `index.html`, `styles.css` y `script.js` a la raíz del repositorio.
3. En GitHub, entra en **Settings**.
4. Entra en **Pages**.
5. En **Build and deployment**, selecciona **Deploy from a branch**.
6. Elige la rama `main` y la carpeta `/root`.
7. Guarda los cambios.
8. GitHub mostrará la dirección pública del sitio.

No necesita servidor ni instalación.
